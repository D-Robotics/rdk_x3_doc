# 地平线RDK套件用户手册

地平线RDK套件用户手册，是旭日X3派用户手册的2.0升级版本。本文档基于RDK X3 2.0版本操作系统，为开发者提供关于RDK X3(X3派、X3模组)产品的使用说明和开发指导，内容涵盖硬件设计、系统定制、应用开发、算法工具链等多个方面。欢迎用户更新体验，具体方法请参考 **快速开始** 章节。
